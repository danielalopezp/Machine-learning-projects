# Machine Learning Projects 🧠

Repositorio que recopila diversos proyectos de *Machine Learning* aplicados a distintos problemas y dominios, como clasificación, regresión y análisis de datos.

## 📂 Estructura

Cada carpeta contiene un proyecto independiente con su propio código, dataset y documentación:

* [Titanic Survival](./Titanic-survival)
* [Spending Score Mall](./Spending-score-mall)
* [Adult Census Income](./Adult-census-income)
* [House Price](./House-price)
* [Ventas Compras](./Ventas-compras)
* [Wine Quality](./Wine-quality)

## 🎯 Objetivo

Explorar, implementar y comparar diferentes técnicas y algoritmos de *Machine Learning*, fomentando el aprendizaje práctico y la experimentación.

## ⚙️ Tecnologías

* Python
* Pandas
* NumPy
* Scikit-learn
* TensorFlow / PyTorch
* Matplotlib / Seaborn

## 🔍 Contenido

Los proyectos incluyen:

* Análisis exploratorio de datos (EDA)
* Entrenamiento de modelos
* Evaluación y optimización
* Visualización de resultados

## 🚀 Uso

Clona el repositorio y accede al proyecto que desees:

```bash
git clone <repo>
cd <nombre-del-proyecto>
pip install -r requirements.txt
python main.py
```

## 📌 Notas

Cada proyecto puede tener dependencias y configuraciones específicas, revisa su README individual para más detalles.
