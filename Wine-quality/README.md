# Wine Quality 🍷

Proyecto de *Machine Learning* enfocado en la predicción de la calidad del vino a partir de sus propiedades físico-químicas.

## 📊 Descripción

Se utiliza un dataset que incluye variables como acidez, pH, nivel de alcohol, densidad y contenido de azúcares. El objetivo es construir un modelo de clasificación o regresión que estime la calidad del vino.

## 🎯 Objetivo

Predecir la calidad del vino y analizar qué características influyen más en su valoración.

## ⚙️ Tecnologías

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 🔍 Proceso

1. Carga y limpieza de datos
2. Análisis exploratorio (EDA)
3. Preprocesamiento de variables
4. Entrenamiento de modelos

## 📈 Resultados

El modelo permite estimar la calidad del vino y comprender la importancia de cada variable en la predicción.

## 🚀 Uso

```bash id="1x8p4m"
pip install -r requirements.txt
python main.py
```

## 📁 Dataset

Dataset de calidad del vino (UCI / Kaggle).
