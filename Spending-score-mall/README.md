# Spending Score Mall 🛍️

Proyecto de *Machine Learning* enfocado en la segmentación de clientes de un centro comercial en función de su comportamiento de gasto.

## 📊 Descripción

Se utiliza un dataset que incluye variables como edad, género, ingresos anuales y *spending score*. El objetivo es aplicar técnicas de *clustering* para identificar grupos de clientes con características similares.

## 🎯 Objetivo

Segmentar a los clientes para comprender mejor sus patrones de consumo y apoyar estrategias de marketing.

## ⚙️ Tecnologías

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 🔍 Proceso

1. Carga y limpieza de datos
2. Análisis exploratorio (EDA)
3. Selección de variables
4. Aplicación de algoritmos de clustering (K-Means)
5. Visualización e interpretación de clusters

## 📈 Resultados

Se identifican distintos perfiles de clientes según su nivel de ingresos y hábitos de gasto, facilitando la toma de decisiones comerciales.
