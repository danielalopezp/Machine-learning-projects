# House Price Prediction 🏠

Proyecto de *Machine Learning* orientado a predecir el precio de viviendas a partir de características estructurales y de ubicación.

## 📊 Descripción

Se utiliza un dataset de precios de viviendas que incluye variables como superficie, número de habitaciones, ubicación, año de construcción y otras características relevantes. El objetivo es construir un modelo de regresión.

## 🎯 Objetivo

Estimar el precio de una vivienda y analizar los factores que más influyen en su valor.

## ⚙️ Tecnologías

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 🔍 Proceso

1. Carga y limpieza de datos
2. Análisis exploratorio (EDA)
3. Entrenamiento de modelos de regresión
4. Evaluación (MAE, MSE, RMSE, R²)

## 📈 Resultados

El modelo permite predecir precios de viviendas con buena aproximación y entender la importancia de cada variable en la estimación.
