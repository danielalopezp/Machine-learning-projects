# Adult Census Income 💼

Proyecto de *Machine Learning* enfocado en predecir si una persona supera cierto nivel de ingresos anuales (>$50K) a partir de datos demográficos y laborales.

## 📊 Descripción

Se utiliza el dataset **Adult Census Income**, que incluye variables como edad, nivel educativo, ocupación, estado civil y horas trabajadas por semana. El objetivo es construir un modelo de clasificación binaria.

## 🎯 Objetivo

Predecir la probabilidad de que un individuo tenga ingresos altos, identificando los factores más influyentes.

## ⚙️ Tecnologías

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 🔍 Proceso

1. Carga y limpieza de datos
2. Análisis exploratorio (EDA)
3. Entrenamiento de modelos de clasificación
4. Evaluación (accuracy)

## 📈 Resultados

El modelo permite clasificar individuos según su nivel de ingresos y analizar qué variables tienen mayor impacto en la predicción.
