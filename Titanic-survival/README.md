# Titanic Survival Prediction 🚢

Proyecto de *Machine Learning* que analiza el dataset del Titanic para predecir la probabilidad de supervivencia de los pasajeros.

## 📊 Descripción
Se utilizan variables como edad, sexo y clase para entrenar modelos de clasificación capaces de identificar patrones y generar predicciones probabilísticas.

## ⚙️ Tecnologías
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 🔍 Proceso
1. Carga y limpieza de datos  
2. Análisis exploratorio (EDA)    
3. Entrenamiento de modelos  
4. Evaluación de resultados  

## 📈 Resultados
El modelo permite estimar la probabilidad de supervivencia de cada pasajero y analizar los factores más influyentes.
