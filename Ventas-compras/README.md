# Ventas y Compras 📊

Proyecto de *Machine Learning* enfocado en el análisis y predicción de datos de ventas y compras para apoyar la toma de decisiones empresariales.

## 📊 Descripción

Se trabaja con un dataset que incluye información sobre clicks y compras.
El objetivo es analizar patrones de comportamiento y construir modelos predictivos.

## 🎯 Objetivo

Predecir tendencias de ventas, identificar patrones de compra y optimizar la gestión de inventario y estrategia comercial.

## ⚙️ Tecnologías

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 🔍 Proceso

1. Carga y limpieza de datos
2. Análisis exploratorio (EDA)
3. Entrenamiento de modelos de predicción
4. Evaluación e interpretación de resultados

## 📈 Resultados

El proyecto permite detectar tendencias, estacionalidad y comportamientos clave que ayudan a mejorar la planificación de ventas y compras.
